# 使用React模仿大众点评

## 技术栈：
react + react-router4.1 + redux + less + ES6/7 + webpack + fetch + bundle-loader

## 运行（nodejs 6.0+）
```
$ npm run dev (正常编译模式)
$ npm run build （发布生产版本，对代码进行混淆压缩，提取公共代码，分离css文件）
```
访问 > http://localhost:8080
